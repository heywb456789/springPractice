✅ Day 2 - Step 1 개념: 투 포인터 (Two Pointers)
📌 투 포인터란?
* 배열을 앞뒤 또는 양쪽에서 동시에 탐색하는 기법
* 보통 정렬된 배열이나 연속된 구간에서 사용
  ```
int lt = 0, rt = 0;
while (rt < arr.length) {
// 구간 [lt ~ rt] 처리
    if (조건이 만족됨) {
        // 처리하고 다음 구간으로 확장
        rt++;
    } else {
        // 조건이 깨짐 → lt 이동
        lt++;
    }
}
```
